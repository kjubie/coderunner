DB Auth Failure fix:

You need to define a password for user postgres

1. Get container
	docker ps

2. Enter inside of the container
	docker exec -it postgresql-container-local bash

3. Switch User to postgres
	su postgres

4. Start query console
	psql

5. Define password
	ALTER ROLE postgres WITH PASSWORD 'admin';
